
# ASKAI

```shell
./askai help
```
```text
askai is a sample CLI tool written in Go

Usage:
  askai [flags]
  askai [command]

Available Commands:
  chat        Chat with ai
  completion  Generate the autocompletion script for the specified shell
  config      Tool for managing configuration file
  help        Help about any command

Flags:
      --config-file string   config file (default is $HOME/.askai.yaml)
  -h, --help                 help for askai

Use "askai [command] --help" for more information about a command.

```
